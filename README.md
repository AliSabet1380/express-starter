# express starter

This file contain express starter file that set up whit bun, typescript, drizzle-orm and neon, zod and much more.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

To generate schema:

```bash
bun run db:generate
```

To migrate schema

```bash
bun run db:migrate
```

some default schema, route, controller and validation are in project
